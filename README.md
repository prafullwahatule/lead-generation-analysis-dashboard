## 🚀 Project Overview
This project focuses on building a **dynamic Power BI dashboard** for analyzing lead generation performance of digital marketing agencies.

The dashboard helps identify:

- 📍 Top performing cities
- 💰 High value leads
- 📈 Lead quality and volume
- 🏢 Agency performance insights
- 📊 KPI-driven business metrics

This project transforms raw agency data into actionable business insights.

---

## 🎯 Business Objective
The objective of this dashboard is to help businesses and stakeholders:

- Track total leads generated
- Measure lead quality using agency ratings
- Identify top-performing cities
- Detect high-value leads
- Improve decision-making using KPI indicators

---

## 📌 Key KPIs
The dashboard includes the following KPI indicators:

- Total Leads
- Total Agencies
- Average Lead Quality Score
- High Value Leads
- High Value Lead %
- Top Performing City

---

## 📊 Dashboard Features
### 1. Executive Dashboard
- KPI Cards
- Top cities analysis
- Lead quality vs volume scatter plot
- Lead distribution chart
- Dynamic slicers

### 2. Detailed Analysis Page
- Matrix view
- Agency-level detailed table
- City-wise performance breakdown
- Conditional formatting insights

---

## 🛠️ Tools Used
- **Power BI**
- **DAX**
- **Power Query**
- **Excel / CSV Dataset**

---

## 📈 Key Business Insights
- Cities with high lead volume maintain strong quality scores
- A few agencies generate majority of high-value leads
- High ratings combined with high review counts indicate strong trust
- Low-rated agencies highlight improvement opportunities
